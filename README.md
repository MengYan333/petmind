# PetMind
介绍视频：https://www.bilibili.com/video/BV1zMG96SERV/?vd_source=8494b9583edd21f583044d60ad37d6a0

- 桌面 AI Agent — 日程决策引擎，不是日程记录工具。
- 用户只需要说出想法，PetMind 负责理解、拆解、决策、准备。
- **传统 todo 软件**：用户输入 → 记录 → 自己设时间/优先级 → 自己决定做什么
- **PetMind**：用户输入 → AI 理解意图 → 自动拆解步骤 → 分析优先级 → 告诉你现在该做什么 → 提前帮你准备资料

## 功能

- **桌面悬浮猫咪**：透明窗口，始终置顶，可拖拽，根据状态自动切换表情，右键退出
- **自然语言任务**：输入一句话，AI 自动理解意图、判断类型、拆解步骤、估算时长
- **今日规划**：从任务池智能生成时间段日程，习惯自动入选，截止任务优先
- **任务提醒**：根据屏幕使用时长和空闲状态，在合适时机主动提醒
- **每日总结**：晚间自动回顾完成情况，分析效率和行为模式，给出明天建议
- **优先级决策**：综合任务属性、当前时段和用户状态，推荐最该做的一件事
- **对话助手**：与 AI 自然对话管理任务，支持搜索资料和调研主题

## 技术栈

- React + Vite
- Electron (桌面应用)
- DashScope API
- Framer Motion (动画)
