一、产品简介
------------------------------------------------------------
PetMind 是一个"世界感知型桌面宠物"应用（类似QQ宠物），
能感知用户的真实环境（时间、天气、屏幕使用时长），
通过 AI Agent 决策宠物的行为表现。

技术栈：React 19 + Vite 8 + Tailwind CSS + Framer Motion + Electron 42
AI 接口：阿里云 DashScope（qwen3.5-plus-2026-04-20 模型，OpenAI 兼容格式）


============================================================
二、网页版功能（http://localhost:5757）
============================================================

【1. 环境感知系统】
  - 时间感知：获取当前小时、分钟，判断是否深夜（23点后）
  - 天气感知：调用 OpenWeatherMap API 获取天气状况和温度
  - 屏幕时长：监测鼠标/键盘活动，5分钟无操作视为休息，每15秒更新

【2. AI Agent 决策系统】
  - 基于传感器数据，AI 自动决定宠物状态
  - 支持 9 种状态：normal、thirsty、sleepy、exercise、rainy、hot、learning、news、happy
  - 优先级规则：早晨新闻 > 深夜催睡 > 久坐运动 > 下雨 > 高温 > 喝水 > 学习 > 普通
  - 每 10 分钟自动运行一次 Agent 决策
  - 支持工具调用：set_pet_state、fetch_learning_summary、fetch_news_summary

【3. 宠物展示】
  - 基于 emoji 的宠物渲染，每种状态有独特的光晕颜色和动画
  - 支持 sm/md 两种尺寸
  - 9 种视觉状态对应不同的动画效果

【4. 习惯提醒系统】
  - 默认习惯：喝水（2小时）、伸展（1小时）、眼睛休息（1小时）
  - 支持自定义添加/删除习惯
  - 数据持久化到 localStorage
  - 右侧滑出抽屉进行习惯配置

【5. 学习摘要功能】
  - 根据用户设定的学习主题（默认"AI大模型"）生成每日学习摘要
  - 3句话总结，每句不超过30字
  - 每日缓存，避免重复调用 API

【6. 新闻摘要功能】
  - 每日生成 3 条新闻摘要（科技、国际、社会）
  - 早晨 8-10 点自动触发
  - 每日缓存，节省 API 调用

【7. 聊天功能】
  - 可折叠的聊天区域
  - 与宠物进行自由对话
  - 支持最多 50 条消息历史
  - 数据持久化到 localStorage
  - 显示消息时间戳

【8. 成长系统】
  - 4 个成长阶段：蛋 → 小鸡 → 小猫 → 成猫
  - 基于 7 天完成率进阶（0% → 30% → 60% → 85%）
  - 属性值：饥饿度（初始70，每小时衰减3）、心情（初始80，每小时衰减1）
  - 互动操作：喂食（饥饿+10）、梳毛（心情+5）、玩耍（快乐+15）
  - 4 个成就：连续7天、阅读30次、雨天互动、护理20次
  - 所有数据持久化到 localStorage

【9. Agent 日志面板】
  - 右下角固定面板，展示 AI 推理过程
  - 可展开/折叠
  - 颜色编码：绿色=工具调用，黄色=决策，白色=信息
  - 思考中显示旋转齿轮动画

【10. 传感器状态栏】
  - 顶部显示当前天气、温度、屏幕使用时长

三、桌面版功能（Electron 浮窗）
============================================================

【1. 桌面浮窗】
  - 透明无边框窗口，始终置顶
  - 全屏工作区域，猫咪可自由拖拽
  - 鼠标穿透：非交互区域自动穿透，不影响桌面操作
  - 自定义 petmind:// 协议，支持从网页版启动桌面版

【2. 猫咪形象】
  - 使用 PNG 图片展示猫咪
  - 支持拖拽移动位置
  - 点击猫咪弹出气泡菜单

【3. 气泡菜单（三标签页）】
  - Today 标签：展示今日新闻摘要
  - Remind 标签：习惯提醒列表
  - Chat 标签：与宠物聊天

【4. 已知问题】
  - 习惯提醒使用硬编码数据，未接入 useHabits hook
  - 聊天记录未持久化，关闭气泡后丢失
  - 新闻内容未持久化，重启 Electron 后丢失
  - 气泡高度固定，聊天消息多时显示拥挤
  - Agent 日志面板未接入（addLog 为空函数）

【5. 已开发但未使用的组件】
  - CatSVG：精细的 SVG 猫咪组件（穿卫衣的灰色猫）
  - CatPopup：悬停弹窗组件（喂食/喝水/梳毛/玩耍）
  - useCatWalk：猫咪行走状态机（坐下/行走/偷看）

五、启动方式
============================================================

网页版：
  cd ~/Documents/petmind
  npm run dev
  浏览器打开 http://localhost:5757

桌面版：
  cd ~/Documents/petmind
  npm run electron:dev

一键启动（Vite + Electron）：
  cd ~/Documents/petmind
  npm run start

六、项目定位
============================================================

PetMind 定位为面试作品集项目，重点展示：
  - AI Agent 决策能力（不是简单聊天机器人，而是带工具调用的 Agent）
  - 传感器数据驱动的智能决策
  - 优先级推理和多工具协调
  - 完整的产品思维（习惯养成 + 学习助手 + 新闻资讯）
